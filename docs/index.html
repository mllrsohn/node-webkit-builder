nw-builder

Build NW.js applications for Mac, Windows and Linux.

For version 3, please go to the corresponding branch.

Major Features

  • Get, run or build applications.
  • Integrate FFmpeg community builds
  • Configure executable fields and icons
  • Downloading from mirrors
  • Node Native Addon support

Installation

With npm:

npm i -D nw-builder

With yarn:

yarn add -D nw-builder

With pnpm:

pnpm add -D nw-builder

Usage

You can use this package via CLI or JavaScript module. To customise your configuration options, please consult the documentation.

CLI

Download NW.js binary

nwbuild --mode=get

Run NW.js application

nwbuild --mode=run

Build NW.js application

nwbuild --mode=get

Module

ESM import

import nwbuild from "nw-builder";

CJS import

import nwbuild from "nw-builder";

let nwbuild = undefined;

nwbuild = await import("nw-builder").default;

Download NW.js binary

nwbuild({
  mode: "get",
});

Run NW.js application

nwbuild({
  mode: "run",
});

Build NW.js application

nwbuild({
  mode: "build",
});

Alternatives

  • nw-builder-platforms - Fork of this repo with platform specific build options
  • nwjs-builder-phoenix - Previously the most used build tool, however it is no longer maintained
